# Scoring Model

## Suggested categories

100 points total:

- 20 ERC-8004 identity and metadata
- 15 endpoint reachability and domain proof
- 15 MCP/A2A service conformance
- 15 x402/Celo payment readiness
- 10 Self Agent ID / sybil-resistance signal
- 10 Celo mainnet activity and contract evidence
- 10 reproducibility, report integrity, and attestation
- 5 docs, open-source metadata, and remediation clarity

## Severity levels

- Critical: prevents another agent from discovering/calling/paying.
- High: makes claims unverifiable or unsafe to rely on.
- Medium: weakens trust or interoperability.
- Low: documentation or metadata quality issue.

## Output fields

- check id
- title
- status: pass / warn / fail / skipped
- severity
- observed evidence
- remediation
- source URL or chain reference
