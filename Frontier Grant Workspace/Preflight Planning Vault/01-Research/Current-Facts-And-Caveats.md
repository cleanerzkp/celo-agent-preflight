# Current Facts and Caveats

## Facts to cite in application

- Prezenti Frontier Pool funds AI and agent economy infrastructure on Celo.
- Frontier asks for working Celo mainnet deployment, infrastructure focus, technical credibility, ERC-8004/Self Agent ID compliance, and verifiable onchain activity.
- Celo docs list AI Agents as a real-world use case and name ERC-8004 for agent trust and x402 for agent payments.
- ERC-8004 is still marked Draft, but it specifies Identity, Reputation, and Validation registries.
- ERC-8004 registration files include services such as web, A2A, MCP, OASF, ENS, DID, email, x402Support, active, registrations, and supportedTrust.
- 8004scan already exposes discovery, leaderboard, registered agents, and Celo examples such as Toppa and Agentic Eye.

## Caveats

- Do not claim Preflight is a security audit.
- Do not claim Preflight guarantees agent honesty or output correctness.
- Do not overstate x402 hosted facilitator support on Celo without proving the exact facilitator route.
- Do not make the project look like a generic scanner. It must generate Celo onchain evidence and be useful to other agents.
