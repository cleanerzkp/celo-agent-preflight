# Submission Checklist

## Public assets

- Product website
- X handle
- KarmaGAP profile
- GitHub repo
- open-source license
- technical docs
- logo
- demo video
- public report examples

## Celo / agent requirements

- working Celo mainnet deployment
- Celo L2 grant delivery address
- ERC-8004 registration or AI-infrastructure exemption explanation
- Self Agent ID status
- verifiable Celo transaction hashes
- attestation contract verified

## Application narrative

- explain infrastructure focus clearly
- show why existing explorers are complementary, not competitors
- prove technical credibility with architecture and docs
- show activation path to first 10 and 100 users
- show concrete ecosystem contribution
- avoid unsupported claims about x402 facilitator support

## KYC / legal

- lead applicant name
- lead applicant email
- company country
- agreement to grant policies
- confirm no grant spend on excluded items such as salaries-only, pure marketing, events, liquidity, token listings, or VC-style funding
