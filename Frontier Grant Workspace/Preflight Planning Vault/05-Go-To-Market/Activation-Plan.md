# Activation Plan

## First 10 users

Target existing Celo agent builders and Frontier applicants. The wedge is free value: a readiness report plus a badge and a concrete fix list.

ReadyList makes the outreach tangible: each builder gets not only a report URL, but a public directory entry showing their agent's current readiness state, evidence, and remediation.

Steps:

1. Identify visible Celo agents.
2. Run preflight manually.
3. Publish public reports.
4. Add each report to ReadyList.
5. DM/email builders with one concrete issue and one quick win.
6. Offer a pull request to add the badge/report/ReadyList link to their docs or metadata.
7. Ask for permission to list them as early users.

## First 100 agents

- Auto-index Celo ERC-8004 registrations daily.
- Generate draft reports.
- Publish ReadyList as the default public index for latest reproducible readiness state.
- Mark stale reports when scans age out or endpoints drift.
- Let builders claim and rerun reports.
- Add GitHub Action for PR checks.
- Publish "Celo Agent Ready" badge.
- Partner with DevRel/hackathons as a launch checklist.

## Distribution proof to collect

- screenshots of outreach
- public GitHub PRs/issues
- badge links in repos
- ReadyList entries created and filtered
- report page analytics
- number of reports generated
- number of Celo attestations written
- number of agents with fixed metadata/endpoints after report
